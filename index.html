Front-end розробник

Про мене:

Front-End розробник із досвідом створення сучасних, адаптивних веб-інтерфейсів. Володію HTML, CSS, SCSS, JavaScript, TypeScript та React, приділяю увагу деталям і якості коду.

Розробляю компонентні бібліотеки та Twig-шаблони для WordPress і ACF, будую UI-системи які зручно підтримувати і масштабувати. В арсеналі — jQuery, Git, SVG-анімації та нестандартні CSS-рішення.

Захоплює все що рухається і дивує — CSS-анімації, інтерактивні ефекти, AI-інструменти у розробці. Люблю коли інтерфейс не просто працює, а відчувається.

З 2024 року працюю як фрилансер — беру проєкти де можна зробити щось справді цікаве.

У своїх проектах я використовую PHP + Twig як шаблонізатор — компонентна система де кожен блок це окремий файл з власними полями. Дані передаються через масиви конфігурації, що дозволяє легко керувати контентом без прив`язки до CMS. Локальне середовище розгорнуто через Laragon.

Стилізація побудована на SCSS з BEM-подібною структурою — кожен компонент має свій файл, змінні через CSS Custom Properties та адаптивна типографіка через clamp(). Компіляція і автопрефіксер через Prepros.

CSS без JavaScript — там де можливо використовую нативні можливості браузера: акордіони, табси, слайдери та анімації появи елементів. JavaScript підключається тільки там де CSS не справляється.

Анімації та інтерактив — SVG спрайти, keyframe анімації для ілюстрацій, хитрощі з grid для плавної зміни контенту без стрибків висоти, flip-картки через 3D трансформації.
